# GitHub Education Igénylése

> **FONTOS**: Olyan eszközt használj, amivel tudsz fotókat készíteni.

Az alábbi lépéseken végighaladva igényelheted a GitHub Student Developer Packet.

## 1. Jelentkezz be a GitHubra GitHubra

1. Nyisd meg a https://github.com oldalt.
2. Jelentkezz be a GitHub fiókodba, vagy regisztrálj, ha még nincs.
3. Bejelentkezés után kattints a profilképedre a jobb felső sarokban.

![Kattints a profilképedre](/assets/guides_md/education/pfp-pic-click.png)

## 2. Beállítások és számlázás megnyitása

1. A lenyíló menüben kattints a **Settings** menüpontra.

![Beállítások megnyitása](/assets/guides_md/education/settings-click.png)

2. A bal oldali sávban kattints a **Billing and licensing** menüpontra.

![Számlázás rész](/assets/guides_md/education/billing-click.png)

## 3. Fizetési adatok hozzáadása

1. Válaszd a **Payment information** menüpontot.
2. Töltsd ki a mezőket valós adatokkal.

![Fizetési adatok kitöltése](/assets/guides_md/education/payment-info-fillout.png)

3. Győződj meg róla, hogy a **Name** mezőben megadott név megegyezik a **nyilvánosan megjelenített neveddel**.

![Nyilvános név ellenőrzése](/assets/guides_md/education/public-name.png)

## 4. GitHub Education igénylésének indítása

1. Kattints meg az **Education benefits** menüpontra.

![Jelentkezés indítása](/assets/guides_md/education/edu-click.png)

2. Kattints a **Start an application** gombra.

![Jelentkezés indítása gomb](/assets/guides_md/education/start-application.png)

## 5. Adatok kitöltése

1. A felugró ablakban válaszd a **Student** lehetőséget.
2. A **School** mezőbe írd be, hogy **"Hack Club"**.
3. Győződj meg róla, hogy az email cím megegyezik a **HC Auth** fiókodhoz tartozó email címmel.
	- Alapértelmezetten ez az az email, amivel a GitHub fiókodat létrehoztad.
	- Szükség esetén hozzáadhatsz másodlagos email címet is.

![Diák jelentkezési ablak](/assets/guides_md/education/first-popup.png)

Ezután kattints a **Share location** gombra, majd a **Continue** gombra.

## 6. Igazolás típusának kiválasztása

1. A következő oldalon kattints a **Select** gombra.

![Select gomb](/assets/guides_md/education/select-proof-type.png)

2. A legördülő menüből válaszd ki, milyen igazolást szeretnél használni.

![Igazolás típusának kiválasztása](/assets/guides_md/education/proof-type-dropdown.png)

> Tipp: A diákigazolvánnyal a legvalószínűbb, hogy jóváhagynak. De nekem az is működött, hogy a Hack Club Slackről készítettem képernyőképet (ehhez a listában az **Other** lehetőséget választottam).

3. Készítsd el a fényképet / képernyőképet, majd erősítsd meg a feltöltést.

## 7. Jelentkezés elküldése

Végül kattints a **Submit application** gombra.

![Jelentkezés elküldése](/assets/guides_md/education/submit-application.png)

Készen vagy! Most már csak meg kell várnod, hogy a GitHub elbírálja és jóváhagyja a jelentkezésed.
